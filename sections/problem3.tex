
\section{问题三：单场竞技最优策略模型}

\subsection{问题分析}

问题二解决的是“给定来袭攻击下如何选择最佳防守动作”的局部匹配问题；问题三则进一步关注单场对抗中的全局决策，即在连续回合内如何组织攻击与防守动作以提升最终胜率。因而，问题三不再是单步评分任务，而是典型的\textbf{序贯决策优化问题}：机器人需要依据双方状态持续决策，在未来多步累计回报意义下实现最优。

在机器人格斗中，当前动作不仅决定即时得失，还会改变后续姿态、稳定性、反击窗口与可选动作集合。例如，高伤害腿法虽具较高输出，却可能增加恢复负担与失稳风险；部分防守动作虽不直接得分，却可显著改善下一步反击条件。因此，问题三应以“长期回报最大化”为核心，而不能仅依据即时收益进行贪心决策。

基于此，本文将单场机器人对抗过程建模为一个有限时域马尔可夫决策过程（Markov Decision Process, MDP），并利用价值迭代求解不同风险偏好下的最优策略。

\subsection{MDP建模}

\subsubsection{状态空间}

设比赛过程在离散时刻上推进，每一决策时刻的系统状态由己方状态、对手状态以及剩余决策阶段共同决定。定义状态为
\[
x=(s,t,\tau),
\]
其中：
\begin{itemize}
    \item \(s\in\mathcal{S}\) 表示己方当前状态；
    \item \(t\in\mathcal{S}\) 表示对手当前状态；
    \item \(\tau\in\{0,1,\dots,T\}\) 表示剩余决策步数或剩余比赛阶段。
\end{itemize}

这里状态集合 \(\mathcal{S}\) 延续前文动作库中的五状态定义：
\[
\mathcal{S}=\{0,1,2,3,4\},
\]
即
\[
0=\text{Idle（空闲）},\quad
1=\text{Guard（防御稳定）},\quad
2=\text{MinorUnstable（轻度失稳）},
\]
\[
3=\text{MajorUnstable（重度失稳）},\quad
4=\text{Down（倒地）}.
\]

因此，状态 \(x=(s,t,\tau)\) 可以完整刻画“双方当前姿态状态 + 比赛剩余时域”的信息，满足马尔可夫性假设：未来转移与收益仅依赖当前状态和当前动作，而与更早历史无关。

\subsubsection{动作空间}

在每个决策时刻，己方可采取的动作包括攻击动作和防守动作两类。记己方动作集合为
\[
\mathcal{A}(s)=\mathcal{A}_{\text{atk}}(s)\cup \mathcal{A}_{\text{def}}(s),
\]
其中：
\begin{itemize}
    \item \(\mathcal{A}_{\text{atk}}(s)\) 表示在当前状态 \(s\) 下合法可执行的攻击动作集合；
    \item \(\mathcal{A}_{\text{def}}(s)\) 表示在当前状态 \(s\) 下合法可执行的防守动作集合。
\end{itemize}

这一定义与问题一和问题二保持一致：攻击动作的可行性由动作库中各攻击的 \texttt{legal\_from} 决定，防守动作的可行性由问题二中的防守合法性约束决定。这样，问题三中的策略优化实际上建立在问题一的攻击动力学分析与问题二的防守匹配结果之上。

\subsubsection{对手策略建模}

由于比赛中对手也会进行动作选择，本文将对手视为一个随机策略执行者，其策略记为
\[
\pi_{\text{opp}}(a_{\text{opp}}\mid t),
\]
表示当对手处于状态 \(t\) 时，选择动作 \(a_{\text{opp}}\) 的概率分布。

这里采用“已知随机策略”假设，原因在于问题三重点在于己方最优决策求解，而非双智能体博弈均衡求解。该假设可视为对手动作模式已通过历史对战数据、仿真库或经验规则预估得到。在实现上，可以令对手更倾向于选择高命中、高收益或低风险动作，从而构成一个有实际意义的基准对手。

\subsubsection{状态转移机制}

给定当前状态 \(x=(s,t,\tau)\)、己方动作 \(a\in\mathcal{A}(s)\) 和对手动作 \(a_{\text{opp}}\in\mathcal{A}(t)\)，系统将在下一时刻转移到新状态
\[
x'=(s',t',\tau-1).
\]

记转移概率为
\[
P(x'\mid x,a,a_{\text{opp}})
=
P(s',t',\tau-1\mid s,t,\tau,a,a_{\text{opp}}).
\]

该转移概率由以下三部分共同确定：
\begin{enumerate}
    \item \textbf{攻击动作动力学影响}：由问题一构建的攻击参数库给出，包括攻击命中概率、攻击后对己方与对手状态的影响、动作稳定性和恢复时间等；
    \item \textbf{防守动作防护效果}：由问题二构建的防守参数库给出，包括完全防御概率、减伤概率、反击窗口概率和防守后状态转移；
    \item \textbf{双方动作耦合机制}：即己方动作和对手动作同时发生时，两者如何叠加影响状态转移。例如，当己方选择攻击而对手选择高质量防守时，对手状态恶化程度将被削弱；若己方选择防守而对手发动高后摇攻击，则己方有机会在下一阶段进入更优反击状态。
\end{enumerate}

因此，问题三中的状态转移并非人为设定，而是由问题一和问题二所建立的随机参数表与动作匹配表共同驱动。

\subsection{奖励函数设计}

问题三的目标不是简单最大化单步输出，而是在整场比赛中取得更高胜率。因此，奖励函数需要同时体现“进攻收益”和“风险控制”两个方面。

设在状态 \(x=(s,t,\tau)\) 下采取动作 \(a\) 后的即时奖励为
\[
R(s,t,a)=\Delta_{\text{harm}}(t,a)-\lambda\cdot \mathbf{1}_{\text{self\_unstab}},
\]
其中：
\begin{itemize}
    \item \(\Delta_{\text{harm}}(t,a)\) 表示己方动作对对手造成的即时伤害或状态压制收益；
    \item \(\mathbf{1}_{\text{self\_unstab}}\) 是指示函数，当己方因当前动作进入轻度失稳、重度失稳或倒地状态时取 1，否则取 0；
    \item \(\lambda>0\) 为风险惩罚系数，用于控制对“高收益高风险动作”的容忍程度。
\end{itemize}

上述奖励函数的含义是：机器人在追求对对手造成更大压制和伤害的同时，避免因自身动作过于激进而导致姿态失稳，进而在后续回合陷入被动。

进一步地，为增强奖励函数对竞技逻辑的刻画，可将其扩展写为
\[
R(s,t,a)
=
\alpha_1 \Delta_{\text{harm}}
+\alpha_2 \Delta_{\text{ctrl}}
+\alpha_3 \Delta_{\text{counter}}
-\alpha_4 \Delta_{\text{risk}},
\]
其中：
\begin{itemize}
    \item \(\Delta_{\text{harm}}\) 表示对手受击伤害收益；
    \item \(\Delta_{\text{ctrl}}\) 表示将对手推入失稳、倒地或压制状态的控制收益；
    \item \(\Delta_{\text{counter}}\) 表示创造反击窗口的机会收益；
    \item \(\Delta_{\text{risk}}\) 表示自身失稳、倒地或高代价恢复的风险损失。
\end{itemize}

在实际实验中，为保持参数简洁，本文采用前述简化形式作为主奖励函数，而将压制收益与反击机会收益通过状态转移概率隐含体现于长期价值函数之中。

\subsection{最优价值函数与贝尔曼方程}

对于任意状态 \(x=(s,t,\tau)\)，记其最优价值函数为 \(V_\tau(s,t)\)，表示在剩余 \(\tau\) 步决策下，从状态 \((s,t)\) 出发所能获得的最大期望累计回报。

则其满足如下贝尔曼最优方程：
\[
V_\tau(s,t)
=
\max_{a\in\mathcal{A}(s)}
\sum_{a_{\text{opp}}}
\pi_{\text{opp}}(a_{\text{opp}}\mid t)
\sum_{s',t'}
P(s',t'\mid s,t,a,a_{\text{opp}})
\Big[
R(s,t,a)+\gamma V_{\tau-1}(s',t')
\Big],
\]
其中 \(\gamma\in(0,1]\) 为折扣因子，用于衡量未来收益相对于当前收益的重要程度。

当 \(\tau=0\) 时，定义终止价值为
\[
V_0(s,t)=0.
\]

因此，问题三的求解本质是在比赛时域内，对每个状态统一比较各候选动作的“即时收益 + 折扣未来收益”，并据此选取最优动作。

\subsection{价值迭代算法}

由于状态与动作空间均为有限集合，本文采用有限时域价值迭代算法求解最优价值函数与最优策略。算法从终端时刻反向递推，逐步计算每个状态下的最优价值，并最终恢复最优动作选择规则。

\begin{algorithm}[H]
\caption{单场竞技最优策略价值迭代}
\label{alg:mdp_vi}
\begin{algorithmic}[1]
\Require 转移模型 \(P\)，对手策略 \(\pi_{\text{opp}}\)，折扣因子 \(\gamma\)，收敛阈值 \(\epsilon\)
\Ensure 最优价值函数 \(V\) 及最优策略 \(\pi^\ast\)

\State 初始化 \(V(s,t)\gets 0\)，对所有状态 \((s,t)\)
\Repeat
    \State \(\Delta\gets 0\)
    \For{每个状态 \((s,t)\)}
        \State \(v\gets V(s,t)\)
        \State
        \[
        V(s,t)\gets
        \max_{a\in\mathcal{A}(s)}
        \sum_{a_{\text{opp}}}\pi_{\text{opp}}(a_{\text{opp}}\mid t)
        \sum_{s',t'}
        P(s',t'\mid s,t,a,a_{\text{opp}})
        \big[
        R(s,t,a)+\gamma V(s',t')
        \big]
        \]
        \State \(\Delta\gets \max(\Delta,|v-V(s,t)|)\)
    \EndFor
\Until{\(\Delta<\epsilon\)}

\For{每个状态 \((s,t)\)}
    \State
    \[
    \pi^\ast(s,t)
    \gets
    \arg\max_{a\in\mathcal{A}(s)}
    \sum_{a_{\text{opp}}}\pi_{\text{opp}}(a_{\text{opp}}\mid t)
    \sum_{s',t'}
    P(s',t'\mid s,t,a,a_{\text{opp}})
    \big[
    R(s,t,a)+\gamma V(s',t')
    \big]
    \]
\EndFor

\State \Return \(V,\pi^\ast\)
\end{algorithmic}
\end{algorithm}

由于有限状态、有限动作的 MDP 在折扣因子 \(\gamma<1\) 时对应的 Bellman 算子具有压缩映射性质，因此价值迭代过程能够收敛到唯一最优值函数。即使在有限时域设置下，该算法也可通过反向动态规划稳定求得最优策略。

\subsection{风险偏好与激进--保守策略族}

在机器人竞技中，不同战术偏好往往对应不同的决策风格。若更强调进攻压制与快速终结，则应容忍一定自身失稳风险；若更强调稳定与容错，则应减少高风险高输出动作的使用频率。本文通过风险惩罚参数 \(\lambda\) 对这两类风格进行统一刻画：

\begin{itemize}
    \item 当 \(\lambda\) 较小时，模型更偏向选择高伤害、高收益动作，即形成\textbf{激进策略}；
    \item 当 \(\lambda\) 较大时，模型更倾向于规避自身失稳，即形成\textbf{保守策略}。
\end{itemize}

通过对不同 \(\lambda\) 取值下的策略进行实验比较，可以得到一条“输出收益--风险控制”的帕累托前沿，从中选择综合胜率最高的折中策略，作为实际推荐的单场最优作战方案。

\subsection{最佳作战决策方案的结构性解释}

基于上述模型，问题三的最优策略通常并非“固定重复某一动作”，而呈现显著的状态依赖性。其典型规律如下：

\analysisparagraph{1. 双方均处于稳定状态时，优先选取高性价比起手动作。}
当己方和对手均处于 Idle 或 Guard 状态时，最优策略往往优先选择命中率较高、恢复代价较低、可继续衔接后续动作的稳定型攻击作为试探或起手动作，而不是直接使用高风险大动作。这样既能维持进攻节奏，又不至于因一次失误导致自身失稳。

\analysisparagraph{2. 对手出现轻度或重度失稳时，应及时切换为高收益压制动作。}
当对手已进入 MinorUnstable 或 MajorUnstable 状态时，其防守能力与规避能力下降，此时价值函数通常会更偏向选择伤害更高、致倒概率更大的动作，以提高终结效率。这体现了 MDP 模型对“机会窗口”的利用能力。

\analysisparagraph{3. 己方失稳时应优先恢复而非盲目反击。}
当己方进入轻度失稳或重度失稳状态时，若仍强行发动高输出攻击，往往会因自身稳定性不足而在未来几个阶段承受更大风险。因此，最优策略通常会在此时优先选择防守、步点调整或平衡恢复类动作，以恢复到可控状态后再重新进入进攻节奏。

\analysisparagraph{4. 倒地状态下策略目标转为“减少失分 + 尽快恢复站立”。}
当己方进入 Down 状态时，最优动作不再是攻击，而是优先采取倒地防御与快速起身，以减少被补击和持续失分的风险。这与问题二中倒地场景的最佳防守匹配结果相一致。

\analysisparagraph{5. 激进策略与保守策略的差异主要体现在高风险腿法和组合技的使用频率上。}
激进策略更可能在中立状态下提前采用高伤害腿法或组合技，以争取快速压制；保守策略则更倾向于先通过稳定型攻击和高质量防守逐步积累优势，待对手出现明显破绽后再进行终结。

\subsection{本问题结论}

本文将单场机器人竞技对抗建模为有限时域 MDP，并对双方联合状态、己方攻防动作、对手随机策略以及动作耦合转移机制进行统一刻画，从而建立单场作战的最优控制模型。该模型能够在每个状态下综合权衡动作的即时收益与长期影响，输出状态相关的最优策略。

模型结果表明，最优策略具有显著的阶段性与状态依赖性：在中立阶段应优先采用命中率高、风险低的稳定型动作建立优势；在对手失稳时应果断切换为高收益压制动作；在自身失稳时应及时恢复姿态，避免因盲目进攻导致被反制；在倒地状态下则应以地面防守和快速起身为首要目标。通过调节风险惩罚参数 \(\lambda\)，模型还可生成从激进到保守的一族策略，并据此选择综合胜率最高的折中方案。


\subsection{模型说明与后续改进方向}

需要指出的是，本文当前采用的是“己方优化、对手随机”的单智能体决策框架，因此重点在于求解对给定对手策略的最优响应，而未进一步求解双方同时理性优化时的博弈均衡。该设定适合用作单场竞技赛中己方策略设计的第一步，但若希望更真实地刻画高水平对抗，则可在后续研究中将模型拓展为双智能体随机博弈或部分可观测博弈模型。

此外，当前状态空间主要刻画双方离散姿态状态，尚未显式引入距离、朝向、擂台位置、电量消耗等连续变量。未来可在现有 MDP 框架基础上进一步扩展状态维度，构建更精细的作战策略优化模型，以提升策略在真实机器人竞技场景中的解释力与适用性。
